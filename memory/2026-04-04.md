# Just a Child - Game Dev Context

## Tech Stack
- Unity 2D, URP, Cinemachine 2.10.7, DOTween, UniRx, Zenject DI, TMPro

## Architecture
- Zenject MonoInstaller: GameInstaller binds all singletons
- PlayerModel (ReactiveProperty fear), PlayerConfig (ScriptableObject), PlayerFearController (coroutines)
- InputController fires OnInteractionPerformed (Subject<Unit>) when interact pressed + interaction available
- PlayerView/IPlayerView: ShowInteraction(bool) toggles interact prompt

## Key Scripts & Paths
- Assets/Scripts/GameFlowController.cs — orchestrates intro (Cinemachine vcam priority swap), 5min timer, win/lose, scene restart
- Assets/Scripts/Furniture/FearAttraction.cs — activates with sound+shake, passive fear increase, player investigates to deactivate+reduce fear, event OnInvestigated
- Assets/Scripts/Furniture/FearAttractionManager.cs — one active at a time, chain activation after investigate
- Assets/Scripts/Furniture/FearDecreasingLightSource.cs — 30s delay, attract flicker, player enters trigger for fear decrease, reload after use
- Assets/Scripts/Furniture/FearDecreaseArea.cs — shows interaction prompt on trigger enter/exit
- Assets/Scripts/PlayerFearController.cs — fear increase/decrease coroutines, InitialFearDelay before fear starts
- Assets/Scripts/PlayerModel.cs — IncreaseFear/DecreaseFear/IncreaseFearBy/DecreaseFearBy, IsFearMaxed
- Assets/Scripts/Player/PlayerConfig.cs — MaxFearValue, FearDecreaseValue, FearIncreaseValue, FearIncreaseInterval, FearDecreaseCooldown, InitialFearDelay
- Assets/Scripts/Player/InputController.cs — reads Interact input, fires OnInteractionPerformed
- Assets/Scripts/UI/TimerView.cs — MM:SS display
- Assets/Scripts/UI/IntroTextView.cs — typewriter text animation
- Assets/Scripts/Installers/GameInstaller.cs — Zenject bindings

## Current Game Flow
1. Awake: introVCam priority=20, playerVCam=0
2. Start: disable movement, hide UI, play intro text (typewriter)
3. OnIntroTextFinished: swap vcam priorities, StartGame()
4. StartGame: enable movement, show fear UI + timer, start fear increase, start attraction spawning
5. FearAttractions activate one at a time, player investigates to deactivate
6. FearDecreasingLightSource available after 30s with attract flicker, reloads after use
7. Win if survived 5min, Lose if fear maxed
8. Both restart scene after delay
